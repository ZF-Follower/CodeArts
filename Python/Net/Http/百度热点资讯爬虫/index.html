而今天才看到微信语音信息，一个在公安系统的朋友反复发我的微信，说是不是本人，我一开始还觉得不可思议，但是现在才知道，这是源于他的职业敏感度，他还以为我的声音被克隆了！

但是今天看到有人居然利用微信语音诈骗，让人觉得不可思议，同时让人警觉。

最近有位赵女士，突然接到自己爸爸微信上发来的消息，父亲称自己出来买菜忘记带钱，让赵女士转200元到微信上，赵女士问了一句“爸爸是你吗？”发个语音给我。

很快收到了回复，赵女士一听就是自己父亲的声音，变想也没想就把钱转了过去，结果被骗了，经过调查发现，赵女士的父亲微信被盗用了，语音是提前录好的，据分析，骗子盗取微信号以后，通过“克隆微信”软件，并用克隆的微信号实施诈骗，这个克隆微信能转发你以前发的微信聊天记录中的语音，还能随意删减好友等等。

如今，语音转发，自动抢红包，自动收账，全球定位，防止对方撤回消息，检查僵尸粉，广告监测等功能都可以实现，但是微信官方没有这些功能们，如此强大的微信其实是一种山寨软件。

如今， 诈骗手段花样百出，大家一定要擦亮双眼，提高警惕，比如收到朋友微信说因刷机通讯录丢失，让你发手机号给他时，要留心，当他进一步发短信给你，让告知验证码，就请一定要注意，因为微信支付有可能被盗用，微信好友被人骗钱，都可能源于这个操作，建议大家涉及财产安全，就要打电话核实，不要给骗子可乘之机。
